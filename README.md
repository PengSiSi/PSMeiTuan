## 实现和内容

在这个React Native仿"美团"项目中,你可以看到以下内容:
1>美团页面效果实现,学习基础组件;
2>React Native项目的目录结构;
3>对于项目中常用的组件,我都进行了封装,方便后续项目的使用;
4>代码规范;
5>添加常用功能的实现的Demo,方便不了解的同学学习和自己以后的使用;
